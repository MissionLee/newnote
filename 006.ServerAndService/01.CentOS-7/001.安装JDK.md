安装JDK

- 查询是否存在yum安装的java包

    **[最小化安装系统应该没有java]**
    ```sh
    rpm -qa | grep java
    ```
    注意：存在yum安装的rpm包可能导致Sqoop无法启动
    - 删除命令  rpm -e --nodeps
    - 这里删除 java开头的就可以了

- 安装对应的JDK

    ```sh
    tar -zxvf  jdk-8u121-linux-x64.tar.gz
    mkdir /usr/java
    mv jdk1.8.0_121/ /usr/java
    ```

- 配置jdk环境变量

    ```sh
    vim  /etc/profile
    ```
    添加如下内容
    ```sh
    export JAVA_HOME=/usr/java/jdk1.8.0_121
    export PATH=$PATH:$JAVA_HOME/bin
    export  JAVA_JRE=.:$JAVA_HOME/lib:$JRE_HOME/lib
    ```
- 导入java环境变量

    ```sh
    source /etc/profile
    ```

- 检查搞定没有
  - 输入 
  - java -version
